# 时间选择器控件配置

```javascript
{
    type: 'timePicker', // 控件类型设置
    key: 'timeSet', // 参数名
    title: '设置制作时间', // 标题
    defaultValue: '10', // 默认值
    unit: 'second', // 最小时间单位
    min: 10, // 最小值（>=0）
    max: 6039, // 最大值
    step: 1, // 步长
    system12: true, // 十二小时制
    stringValue: true // 最终下发值转换为字符串
}
```
